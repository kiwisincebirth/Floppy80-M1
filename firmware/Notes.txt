--------------------
- CPM 1.4.1 (Lifeboat)
  FORMAT
  - works
  COPY
  - PIP
    at * prompt type "destingation=source"
    for example "C:FDC.COM=B:FDC.COM"

- CPM 1.5 (FGM)
  FORMAT
  - works
  COPY
  - PIP
    at * prompt type "destingation=source"
    for example "C:FDC.COM=B:FDC.COM"

- Double DOS 4.2.4
  FORMAT
  - works
    DOUBLE FORMAT :2
    DOUBLE BACKUP :0 TO :2
  COPY
  - "COPY FILENAME/EXT:0 TO FILENAME/EXT:2"

- DOS Plus 3.50
  FORMAT
  - works, provided you start with a double sides dmk file.
  COPY
  - COPY FDC/CMD:0 FDC/CMD:2
  BACKUP
  - works as long as disk doubler is disabled

- LDOS 5.3.1
  FORMAT
  - works
  - supports only single density at this time
  - requires a reboot after formatting drive
  COPY
  - COPY FDC/CMD:1 FDC/CMD:2

- MultiDOS 4.01
  FORMAT
  - works
  BACKUP
  - works as long as disk doublers are disabled

- NEWDOS 3.0
  FORMAT
  - works
  COPY
  - "COPY FILENAME/EXT:0 TO FILENAME/EXT:2"

- NEWDOS 80 V2.0
  FORMAT
  - works
  COPY
  - COPY FDC/CMD:0 FDC/CMD:2

- TRSDOS 2.3
  FORMAT
  - works
  COPY
  - COPY FDC/CMD:1 FDC/CMD:2

- UltraDOS 4.2.0
  FORMAT
  - works
  COPY
  - COPY FDC/CMD:1 FDC/CMD:2

- VTDOS 3.0.0
  FORMAT
  - works
  COPY
  - COPY FDC/CMD:1 FDC/CMD:2
  BACKUP
  - Different Pack IDs

- VTDOS 4.0.1
  FORMAT
  - works
  COPY
  - COPY FDC/CMD:1 FDC/CMD:2
  BACKUP
  - works as long as disk doubler is disabled

- SB-M1-RPM
  Access FDC by pressing break and entering
  cmd"fdc ini"

Power Up Sequence
--------------------
- Start with the keyboard powered off and
  USB removed from the Floppy80
- Connect USB to the Floppy80
- Power on the keyboard
