
     OS              Boots   Format   Copy   FDC/CMD
-----------------------------------------------------------
Double DOS 4.2.4      Yes      No      Yes    Yes
FMG CPM 1.5           Yes      Yes     Yes    Yes
LDOS 5.3.1            Yes      Yes     Yes    Yes
Lifeboat CPM 1.4.1    Yes      Yes     Yes    Yes
MultiDOS 4.01         Yes      Yes     Yes    Yes
NEWDOS 3.0            Yes      Yes     Yes    Yes
NEWDOS 80 V2.0        Yes      No      Yes    Yes
TRSDOS 2.3            Yes      Yes     Yes    Yes
UltraDOS 4.2.0        Yes      Yes     Yes    Yes
VTDOS 3.0.0           Yes      Yes     Yes    Yes
VTDOS 4.0.1           Yes      Yes     Yes    Yes

--------------------
- Double DOS 4.2.4
  FORMAT
  - do not have a format program
  COPY
  - "COPY FILENAME/EXT:0 TO FILENAME/EXT:2"

- FGM CPM 1.5
  FORMAT
  - works
  COPY
  - PIP
    at * prompt type "destingation=source"
    for example "C:FDC.COM=B:FDC.COM"

- LDOS 5.3.1
  FORMAT
  - works
  - supports only single density at this time
  - requires a reboot after formatting drive
  COPY
  - COPY FDC/CMD:1 FDC/CMD:2

- Lifeboat CPM 1.4.1
  FORMAT
  - works
  COPY
  - PIP
    at * prompt type "destingation=source"
    for example "C:FDC.COM=B:FDC.COM"

- MultiDOS 4.01
  FORMAT
  - works as long as the disk double code in firmware is disabled

- NEWDOS 3.0
  FORMAT
  - works
  COPY
  - "COPY FILENAME/EXT:0 TO FILENAME/EXT:2"

- NEWDOS 80 V2.0
  FORMAT
  - does not work
  COPY
  - COPY FDC/CMD:0 FDC/CMD:2

- TRSDOS 2.3
  FORMAT
  - works
  COPY
  - COPY FDC/CMD:1 FDC/CMD:2

- UltraDOS 4.2.0
  FORMAT
  - works
  COPY
  - COPY FDC/CMD:1 FDC/CMD:2

- VTDOS 3.0.0
  FORMAT
  - works
  COPY
  - COPY FDC/CMD:1 FDC/CMD:2

- VTDOS 4.0.1
  FORMAT
  - works
  COPY
  - COPY FDC/CMD:1 FDC/CMD:2

- SB-M1-RPM
  Access FDC by pressing break and entering
  cmd"fdc ini"

Power Up Sequence
--------------------
- Start with the keyboard powered off and
  USB removed from the Floppy80
- Connect USB to the Floppy80
- Power on the keyboard
